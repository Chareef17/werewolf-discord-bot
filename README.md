# 🐺 Werewolf Discord Bot — เกมหมาป่า

บอท Discord สำหรับเล่นเกมหมาป่า (Werewolf / Mafia) รองรับ 4-16 ผู้เล่น

## บทบาทในเกม

| บทบาท | อิโมจิ | ทีม | ความสามารถ |
|--------|--------|------|------------|
| หมาป่า (Werewolf) | 🐺 | หมาป่า | เลือกกัดผู้เล่น 1 คนต่อคืน |
| หมอดู (Seer) | 🔮 | ชาวบ้าน | ตรวจสอบตัวตนผู้เล่น 1 คนต่อคืน |
| หมอ (Doctor) | 💉 | ชาวบ้าน | ปกป้องผู้เล่น 1 คนต่อคืน |
| นักล่า (Hunter) | 🏹 | ชาวบ้าน | เมื่อตาย ยิงผู้เล่น 1 คนไปด้วย |
| ชาวบ้าน (Villager) | 👨‍🌾 | ชาวบ้าน | โหวตขับไล่หมาป่าตอนกลางวัน |

## การแจกบทบาท

| จำนวนผู้เล่น | หมาป่า | หมอดู | หมอ | นักล่า | ชาวบ้าน |
|-------------|--------|-------|-----|--------|---------|
| 4-5 คน | 1 | 1 | 1 | - | ที่เหลือ |
| 6-7 คน | 2 | 1 | 1 | - | ที่เหลือ |
| 8-9 คน | 2 | 1 | 1 | 1 | ที่เหลือ |
| 10+ คน | 3 | 1 | 1 | 1 | ที่เหลือ |

## เงื่อนไขชนะ

- **ชาวบ้านชนะ**: กำจัดหมาป่าทั้งหมด
- **หมาป่าชนะ**: จำนวนหมาป่า ≥ จำนวนชาวบ้าน

## การติดตั้ง

### 1. สร้าง Discord Bot

1. ไปที่ [Discord Developer Portal](https://discord.com/developers/applications)
2. คลิก "New Application" ตั้งชื่อบอท
3. ไปที่แท็บ "Bot" แล้วคลิก "Add Bot"
4. เปิด **Privileged Gateway Intents**: Message Content Intent
5. คัดลอก **Bot Token**
6. ไปที่แท็บ "OAuth2" > "URL Generator"
   - Scopes: `bot`, `applications.commands`
   - Bot Permissions: `Send Messages`, `Embed Links`, `Use Slash Commands`
7. คัดลอก URL แล้วเปิดในเบราว์เซอร์เพื่อเชิญบอทเข้า Server

### 2. ตั้งค่าโปรเจค

```bash
cd werewolf-discord-bot

# ติดตั้ง dependencies
npm install

# คัดลอกไฟล์ env
cp .env.example .env
```

### 3. แก้ไขไฟล์ .env

```env
DISCORD_TOKEN=your_bot_token_here
```

- `DISCORD_TOKEN` — Bot Token จากขั้นตอนที่ 1 (คัดลอกจากหน้า Bot ใน Developer Portal)

### 4. ลงทะเบียน Slash Commands

```bash
npm run deploy
```

### 5. เริ่มต้นบอท

```bash
npm start
```

## วิธีเล่น

1. พิมพ์ `/werewolf` ในช่อง Discord เพื่อสร้างห้องเกม
2. ผู้เล่นอื่นกดปุ่ม **"เข้าร่วม"** เพื่อเข้าร่วมเกม
3. เจ้าของห้องกด **"เริ่มเกม"** เมื่อพร้อม (ต้องมีอย่างน้อย 4 คน)
4. ทุกคนจะได้รับบทบาทผ่าน DM
5. **จบเกม**: เจ้าของห้องกดปุ่ม **"จบเกม"** ในห้องล็อบบี้ หรือใช้ `/werewolf` แล้วเลือก **จบเกม** ตอนเล่นอยู่ก็ได้

### กลางคืน 🌙
- **หมาป่า**: เลือกคนที่จะกัดใน DM
- **หมอดู**: เลือกคนที่จะตรวจสอบใน DM
- **หมอ**: เลือกคนที่จะปกป้องใน DM

### กลางวัน ☀️
- ดูผลจากกลางคืน (ใครตาย หรือรอดตาย)
- หารือกันในช่องแชท
- โหวตขับไล่คนที่สงสัยว่าเป็นหมาป่า

### เกมจบเมื่อ
- หมาป่าตายหมด → **ชาวบ้านชนะ**
- หมาป่ามีจำนวนเท่ากับหรือมากกว่าชาวบ้าน → **หมาป่าชนะ**

## กฎพิเศษ

- หมอไม่สามารถปกป้องคนเดิมสองคืนติดกัน
- หากโหวตเสมอกัน จะไม่มีใครถูกขับไล่
- นักล่าที่ถูกกำจัด สามารถเลือกยิงผู้เล่น 1 คนก่อนตาย
- ถ้าหมาป่ามีหลายตัว จะโหวตกันเองเลือกเป้าหมาย
